# Между нами

Романтическая анкета для девушки: 20 вопросов без правильных ответов — 10 с вариантами и 10 открытых. Ответы сохраняются в браузере, показываются в финале и отправляются владельцу сайта через системное меню «Поделиться» или копированием текста.

## Публикация на GitHub Pages

1. Создайте пустой репозиторий на GitHub и загрузите в него содержимое этой папки.
2. В репозитории откройте **Settings → Pages**.
3. Выберите **Deploy from a branch**, ветку `main` и папку `/(root)`.
4. Откройте URL, который покажет GitHub, и отправьте его девушке.

Сайт не использует сервер и не собирает данные: чтобы вы увидели ответы, девушка нажимает «Отправить мне ответы» на финальном экране и выбирает ваш мессенджер.
